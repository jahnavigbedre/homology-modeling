# Template Structures

Place PDB template files in this directory.

Required files:
- `1crn.pdb` - Crambin structure (template)

Download from: https://www.rcsb.org/

## Directory Structure

The project directory is organized as follows:

```
homology-modeling/
├── data/
│   └── templates/
├── results/
│   └── figures/
│       └── .gitkeep
├── scripts/
│   ├── align_sequences.py
│   ├── modeller_build.py
│   ├── evaluate_model.py
│   └── analyze_results.py
├── .gitignore
└── requirements.txt
```
